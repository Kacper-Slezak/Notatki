# Kubectl apply

##  Co to jest? 
Komenda która jes twykorzystaywa do aplikowani zmian w plikach yaml do działania

##  Jak to działa?
1. Bierze pod uwagę:
- Plik lokalny
- Ostatnio Zaaplikowaną Konfiguracje
- Konfiguracje działającego obiektu
1. Na tej podstawie decyduje jakie zmiany wprowadziliśmy

##  Code Snippet / Komendy
```bash
kubectl apply -f nazwa_pliku.yaml
```
